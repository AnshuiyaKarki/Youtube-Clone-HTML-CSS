# YouTube-Clone-HTML-CSS (Static Website)

A simple static front-end clone of the YouTube homepage built using HTML and CSS. This project is created for practice purposes to improve layout design skills, Flexbox usage, and UI structuring.

---

## 📌 Features

- YouTube-like header with search bar and icons
- Sidebar navigation (Home, Trending, Subscriptions, etc.)
- Video grid layout with thumbnails
- Category/filter navigation bar
- Responsive layout (basic support depending on CSS)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Google Material Icons

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/Youtube-Clone-HTML-CSS.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional setup is required since this is a static website.
